Welcome to the AnimalGuessingGame. The game is simple, guess the animal type that pops up correctly and earn a point! Rack up as many points as you can. There is a cheat button that will award a point to the score when used. If you feel that you have a match then click the "Is This A Match" button to find out. When you are done, you can press the leaderboard link to put your score on the leaderboard. Have Fun!
